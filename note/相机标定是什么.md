# 相机标定是什么

**相机标定（Camera Calibration）** 是计算机视觉领域中**确定相机固有参数和相机在空间中姿态**的过程，核心目标是建立 **像素坐标系**（图像上的像素点位置）与 **世界坐标系**（真实 3D 空间中的物理位置）之间的精准数学映射关系。

简单来说，相机成像本质是把 3D 空间中的点投射到 2D 图像平面上，这个过程会受到相机自身特性和拍摄姿态的影响，相机标定就是**量化这些影响并建立转换公式**的步骤。

### 一、 相机标定的核心对象：两类关键参数

1. **内部参数（Intrinsic Parameters）**

   这是相机的**固有属性**，由硬件结构决定，一旦相机生产完成就基本固定（除非镜头更换或损坏）。

   

   - **焦距（fₓ, fᵧ）**：相机光学中心到图像传感器的距离，单位通常是像素（与传感器像素尺寸相关）。

   - **主点（cₓ, cᵧ）**：相机光轴与图像传感器的交点，理想情况下是图像中心，实际会有微小偏移。

   - 畸变系数（k₁, k₂, p₁, p₂ 等）：矫正镜头的光学畸变（包括径向畸变：图像边缘的拉伸 / 收缩；切向畸变：镜头安装偏差导致的倾斜畸变）。

     这些参数会被整理为内参矩阵，是像素坐标和相机坐标系转换的核心。

2. **外部参数（Extrinsic Parameters）**

   描述**相机在世界坐标系中的位置和姿态**，是每次拍摄时的可变参数。

   

   - **旋转矩阵（R）**：表示相机坐标系相对于世界坐标系的旋转角度。

   - 平移向量（t）：表示相机光学中心在世界坐标系中的位置坐标。

     外参的作用是实现世界坐标系到相机坐标系的转换。

### 二、 相机标定的核心作用

1. **消除图像畸变**：标定得到的畸变系数可以修正镜头带来的成像失真，让图像更接近真实场景。
2. **实现 2D 到 3D 的转换**：只有通过标定参数，才能根据图像上的 2D 像素点，计算出它在真实 3D 空间中的位置。
3. **支撑多视角 3D 重建**：在你之前提到的 3D 高斯场景重建中，**相机标定产生的稀疏点**就是这样来的 —— 通过多视角图像的标定参数，对同一空间点在不同图像中的投影位置进行**三角化计算**，得到该点的 3D 坐标，这些稀疏点是后续 3D 高斯建模的几何基础。

### 三、 相机标定的常用方法（以棋盘格标定为例）

1. 准备**高精度棋盘格标定板**：棋盘格的方格尺寸是已知的（比如 20mm×20mm），作为世界坐标系的参考。
2. 从**不同角度、不同距离**拍摄多张标定板图像（通常需要 10-20 张）。
3. 提取每张图像中棋盘格的**角点**（2D 像素坐标），并对应到世界坐标系中的已知 3D 坐标（比如棋盘格左上角为原点）。
4. 用标定算法（如张正友标定法）求解内参矩阵、外参矩阵和畸变系数。
5. 验证标定精度：用重投影误差判断标定效果（误差越小，标定越精准）。
