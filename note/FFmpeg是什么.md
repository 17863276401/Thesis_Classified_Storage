# FFmpeg是什么

简单来说，**FFmpeg** 是音视频处理领域的“瑞士军刀”。它是一个开源、跨平台的命令行工具集合，几乎可以处理任何格式的音频、视频和多媒体文件。

无论是你手机上的视频播放器、短视频平台的后台转码，还是直播软件（如 OBS），核心大多都集成了 FFmpeg。

------

### FFmpeg 的核心功能

#### 1. 格式转换 (Transcoding)

这是它最常用的功能。你可以把 `.avi` 转成 `.mp4`，或者把 `.flv` 转成 `.mkv`。它支持上千种编码格式。

- **例子：** 将一个视频文件转换成适合网页播放的 MP4。

#### 2. 视频剪辑与处理

不需要打开复杂的剪辑软件，通过一行命令就能完成：

- **裁剪 (Cropping)：** 切除视频黑边。
- **切片 (Slicing)：** 把 1 小时的视频切成 10 段，或者截取其中 5 秒。
- **调整尺寸 (Resizing)：** 把 4K 视频缩小成 1080P 或 720P。
- **提取音频：** 从电影里直接把背景音乐提取出来存为 MP3。

#### 3. 视频滤镜

它内置了非常强大的滤镜库：

- 添加水印（文字或图片）。
- 调整亮度、对比度、色调。
- 旋转视频、去噪、倍速播放。

#### 4. 直播推流与拉流

FFmpeg 可以读取摄像头、麦克风或屏幕内容，并将其实时编码发送到直播服务器（如 Bilibili、YouTube），也可以从网上抓取直播流保存到本地。

------

### 为什么大家都在用它？

| **特点**     | **说明**                                                     |
| ------------ | ------------------------------------------------------------ |
| **全能性**   | 几乎支持历史上出现过的所有音视频协议和格式。                 |
| **高性能**   | 采用 C 语言编写，针对各种 CPU 指令集（如 AVX, SSE）进行了深度优化，处理速度极快。 |
| **开源免费** | 它是开源项目，开发者可以自由地将其集成到自己的软件中。       |
| **跨平台**   | 在 Windows, macOS, Linux, Android, iOS 上都能完美运行。      |

------

### 举个简单的命令例子

如果你安装了 FFmpeg，在命令行输入以下代码就能把视频转码：

Bash

```
# 将 input.mov 转换为 1080p 的 input.mp4
ffmpeg -i input.mov -s 1920x1080 input.mp4
```
