# 先验正则化是什么

**先验**指模型优化前已知的、可靠的**前置信息 / 常识 / 观测线索**，是引导模型学习的 “参考依据”，避免优化陷入不合理方向。

结合论文场景，核心先验就是：

- 深度先验：传感器（如 iPhone）采集的深度数据、单目网络（ZoeDepth）估计的深度；
- 法向量先验：Omnidata 模型估计的场景表面法向量。

这些信息虽非绝对完美（如 iPhone 深度有噪声），但能反映真实场景的几何规律，是解决无纹理 / 病态场景重建歧义的关键。

**先验正则化**：在模型优化时，引入**已知的、可靠的先验信息**（比如几何规律、外部观测线索）作为约束，引导模型朝着 “符合真实场景逻辑” 的方向学习，避免优化过程中出现歧义、伪影或过拟合。

结合论文场景，核心是用**深度先验**（传感器 / 单目估计的深度）和**法向量先验**（单目网络估计的表面法向量），约束 3D 高斯的位置、朝向和形状 —— 比如让高斯的深度贴合真实场景深度，法向量对齐物体表面真实朝向，从而解决无纹理 / 病态场景的几何约束不足问题。
