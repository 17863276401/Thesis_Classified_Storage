# artifact与aliasing的区别

在技术领域（尤其是你关注的图像处理、点云 / 3D 视觉场景）中，**artifact（伪影）是一个宽泛的 “统称”，而 aliasing（混叠 / 走样）是 artifact 的一种**，二者的核心区别体现在**定义、成因、表现**上，具体对比如下：

### 1. 核心定义

|            术语             |                           核心定义                           |
| :-------------------------: | :----------------------------------------------------------: |
|    **Artifact（伪影）**     | 技术过程中**非预期产生的所有干扰 / 失真特征**的统称，是 “各类非自然 / 非真实缺陷” 的泛指。 |
| **Aliasing（混叠 / 走样）** | 是 Artifact 的**特定子类**，仅因 “采样频率低于信号最高频率（违反奈奎斯特定理）” 导致的失真。 |

### 2. 成因差异

- **Artifact 的成因（多样）**：

  涵盖设备、算法、环境、处理流程等多种因素，例如：

  - 点云：拼接算法误差导致的缝隙、深度相机反光产生的噪点；
  - 图像：压缩算法导致的块效应、镜头畸变导致的变形；
  - 生成模型：GAN 输出的边缘扭曲、纹理错乱。

- **Aliasing 的成因（单一）**：

  仅因**采样不足**：当采样频率低于信号（如图像、点云表面细节）的 2 倍最高频率时，高频信号会被错误地 “折叠” 成低频信号，产生失真。

  

### 3. 典型表现（技术场景）

|     术语     |          典型表现（以你的点云 / 图像处理场景为例）           |
| :----------: | :----------------------------------------------------------: |
| **Artifact** | 点云：拼接冗余点、颜色滤波后的色偏区域；图像：生成式模型的面部变形、视频压缩的马赛克。 |
| **Aliasing** | 点云：低采样密度导致的表面 “锯齿状边缘”；图像：斜线 / 纹理的 “阶梯状锯齿”（如游戏低分辨率下的边缘）、纹理的摩尔纹（重复图案的错乱叠加）。 |

### 4. 所属关系

**Aliasing 是 Artifact 的一个具体类型**—— 所有 aliasing 都属于 artifact，但 artifact 包含更多非 “采样不足” 成因的失真（如算法错误、设备缺陷导致的问题）。

### 5. 解决方法

- **Artifact**：需**针对性处理**（如点云拼接伪影→优化配准算法；生成式伪影→调整模型参数）；
- **Aliasing**：核心是**满足采样条件**（如提高点云采样密度、图像渲染前加抗混叠滤波器（AA）、音频采样前加低通滤波器）。
